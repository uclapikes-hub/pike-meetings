If a meeting was deleted before Stage 4's cascade fix, its no-shows and fines may have been left behind as orphans. This button finds any no-show, fine, attendance record, or absence request whose meeting no longer exists and deletes them.

Safe to run anytime. Only deletes data that has nothing to point to. Cannot be undone.

If the no-show processor created duplicate records for the same brother on the same meeting (e.g., from cache-stale timer ticks), this finds (brother, meeting) pairs with multiple no_show entries, keeps the earliest, and deletes the rest. Auto-recalculates fines. Safe to run anytime.
